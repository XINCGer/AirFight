# AirFight
使用Cocos2dx-3.8&amp;C++开发的雷霆战机小游戏

## 简介  
>* 一款简单但是功能完整的飞机空战小游戏，使用coocs2dx引擎，c++语言开发。  
实现了飞机飞行，飞机碰撞，发射子弹，敌机发射子弹，背景音乐，子弹特效，  
分数统计，菜单管理等功能。
